About GeNest

GeNest (v0.13.14) is a specialized tool developed by the TGGR lab for systematic identification and analysis of nested genes across genomes. The tool analyzes GTF files to detect and characterize genes that are fully contained within other genes, classifying different types of overlaps and generating detailed reports of these genomic arrangements.

Analysis Reports

Each report contains:

  • Distribution and statistics of nested gene types (intronic, exonic, TSS/TTS overlaps)
  • Analysis of strand relationships between host and nested genes
  • Host gene size distributions and nested gene positions
  • Interactive tables of all identified nested gene pairs
  • Host genes with multiple nested genes
  • Genes that are both nested and hosts
