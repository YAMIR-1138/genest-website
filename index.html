About GeNest

Welcome to the GeNest Report Repository - a collection of nested gene analysis reports. These reports are generated using GeNest, a computational tool we designed for systematic identification and analysis of nested genes across genomes.

GeNest analyzes genomic data to detect and characterize genes that are contained within other genes, providing insights into genomic organization and complexity. Each report in this repository contains statistical analysis, visualizations, and downloadable datasets.

Analysis Reports Content

Each report includes:

  • Distribution and statistics of nested gene types
  • Strand relationships between host and nested genes
  • Host gene size distributions and nested gene positions
  • Interactive tables of all identified nested gene pairs
  • Host genes with multiple nested genes
  • Genes that are both nested and hosts
  • Interactive plots and visualizations
