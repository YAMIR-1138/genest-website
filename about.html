About GeNest

Advanced Nested Gene Analysis Platform

Purpose

GeNest is a computational tool designed for detecting and analyzing nested genes in genomic datasets. It identifies genes contained within other genes' genomic regions, providing insights into genomic organization and complexity.

Key Capabilities

Gene Detection

  • Intronic Nested Genes
  • Exonic Overlap Nested Genes
  • TSS/TTS Overlap Genes
  • Complex Overlap Genes

Processing Features

  • Multi-core processing
  • Low RAM mode option
  • Supports coding and non-coding gene detection
  • Organism auto-detection
  • Comprehensive result filtering

Output Types

  • Detailed HTML/PDF reports
  • Interactive data tables
  • Statistical visualizations
  • Distribution plots

Development

GeNest is developed by the TGGR Lab. The project is open source.
